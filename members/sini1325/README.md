# 개인 프로젝트: sini1325

> 이 디렉토리는 **sini1325** 의 개인 작업 공간입니다.

## 프로젝트 설명

- **설문 문항 설계**
  - 사용자의 스타일 성향이 자연스럽게 드러나도록 패션 취향, 상황별 선택, 핏 선호를 반영한 설문 문항을 설계했습니다.
  - 결과 페르소나가 균형 있게 도출되도록 문항별 선택지의 방향성과 표현을 조정했습니다.

- **문항–데이터 매핑 방식 설계**
  - 각 문항의 선택지가 특정 스타일 페르소나와 연결되도록 매핑 규칙을 설계했습니다.
  - 설문 응답이 최종 결과로 이어지는 점수 누적형(rule-based) 구조를 정리해, 결과 산출 로직의 일관성을 높였습니다.
  - 퍼스널컬러, 스타일 키워드, 카테고리/핏 추천 데이터가 연결될 수 있도록 데이터 구조를 기획했습니다.

- **프론트엔드 디자인 기획 및 구체화**
  - 생성형 AI(Claude)를 활용해 서비스 컨셉에 맞는 UI 톤앤매너, 화면 구성, 문구 스타일을 빠르게 탐색하고 구체화했습니다.
  - 테스트형 서비스에 맞게 화면 흐름(시작 → 진단 → 설문 → 결과 → 추천 상품)과 결과 카드 중심의 UX를 설계했습니다.
  - 전반적인 비주얼 방향성(컬러, 카드형 레이아웃, 버튼/태그 스타일, 캐주얼한 카피 톤) 정리에 기여했습니다.
