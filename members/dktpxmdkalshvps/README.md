# Musinsa Personal Color & MAPCTI Recommendation System

이 프로젝트는 사용자의 스타일 성향(MAPCTI)과 퍼스널 컬러 정보를 결합하여 무신사(Musinsa)에서 최적의 코디 아이템을 자동으로 추천해주는 시스템입니다.

## 🎯 프로젝트 목적
사용자가 자신의 패션 스타일과 어울리는 색상을 알고 있더라도, 실제 쇼핑몰에서 해당 조건에 맞는 상품을 일일이 찾는 것은 번거로운 작업입니다. 이 도구는 스타일 유형(MAPSITI)과 **퍼스널 컬러** 데이터를 무신사의 검색 필터와 매핑하여, 사용자에게 최적화된 상품 리스트를 실시간으로 제공합니다.

## 🛠 기술 스택
- **Language**: Python 3.x
- **Libraries**:
  - `undetected-chromedriver`: 무신사의 봇 감지 시스템을 우회하기 위한 동적 브라우저 제어
  - `BeautifulSoup4`: 수집된 HTML 데이터 파싱 및 정보 추출
  - `urllib`: URL 인코딩 및 파라미터 구성

## ✨ 핵심 기능
1. **MAPCTI 스타일 매핑**
   - 사용자의 스타일(미니멀, 캐주얼, 로맨틱, 스포티 등)을 무신사 내부 스타일 코드로 변환하여 정확한 타겟팅이 가능합니다.
2. **퍼스널 컬러 기반 색상 필터링**
   - 12가지 세부 퍼스널 컬러 타입(봄 라이트, 가을 딥 등)별 권장 색상 리스트를 보유하고 있으며, 이를 무신사 검색 엔진의 색상 필터에 적용합니다.
3. **지능형 동적 크롤링**
   - `undetected-chromedriver`를 사용하여 실제 사용자의 브라우징 패턴을 모방, 실시간 인기 판매 순위(SALE_ONE_WEEK_COUNT) 기반 데이터를 수집합니다.
4. **성별 및 카테고리 최적화**
   - 남성/여성별 스타일 제한(예: 남성용 로맨틱 스타일 필터링 등) 및 카테고리별(상의, 바지, 원피스 등) 맞춤형 검색을 지원합니다.
